Personal Projects

Over the past few months, I have been building up my github repository. While it is in its early stages, I have been adding small personal repositories to it, in addition to the collaborative work I have been doing in large scale image classification. This past summer (2018), I wrote an RNN to predict stock prices using financial data from Quandl, and also followed this tutorial on implementing siamese neural networks, except I wrote the model in TensorFlow instead of Keras. I am currently working on a PyTorch implementation of this paper on ladder variational encoders; the original source code written using Theano, Lasagne and Parmesan is available here. This implementation is not quite ready yet; I hope to have it up soon. The code will be based on Homework 2 of Stefano Ermon's newly offered course on Deep Generative Models here at Stanford.
